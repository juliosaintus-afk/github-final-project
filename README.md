## Simple Interest Calculator

A calculator that calculates simple interest given principal, annual rate of interest and time period in years.

## Description
Simple interest is a quick way to calculate the interest charge on a loan or investment. 
It is widely used in micro-finance and personal banking to determine the cost of borrowing 
or the return on an investment over a period of time.

## Formula
simple interest = p*t*r/100

## Input
- `p` - principal amount (the initial amount of money)
- `t` - time period in years
- `r` - annual rate of interest (in percentage)

## Output
- `simple interest` = p*t*r/100

## Example
| Input | Value |
|-------|-------|
| Principal (p) | 1000 |
| Time (t) | 2 years |
| Rate (r) | 5% |
| **Simple Interest** | **100** |

Calculation: 1000 * 2 * 5 / 100 = **100**

## Usage
Run the script using:
```bash
bash simple-interest.sh
```
Follow the prompts to enter the principal, time period, and interest rate.
